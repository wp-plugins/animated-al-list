It helps you to create beautiful dynamic menu(link list/text information/images/icons).It will revive your site and let users look at your page anew.

== Description ==

You can see <a href="http://al-plugins.biz/animated-al-list/">Animated AL List Live Demo</a>. In this demo is shown menu from icon and text elements. But you can create an animated menu from images also.
Animated AL List allows you not only decorate your site active animation but focus users attention on important aspects of your site.

### Main features of the Animated AL List

*   Responsive design.
*   Capability to add images.
*   Capability to add texts.
*   Capability to make buttons.
*   Capability to add links.
*   Capability to manage list elements.
*   Capability to attach css rules to text items and to list.
*   Generate shortcodes to add to pages or posts or template file.
*   You can add a set of css rules to any item.
*   You can add a list as widget.
*   Different types of curves and lines.
*   Different types of effects.
*   Jquery animations and css transitions.
*   Predefined states.
*   Capability to create main and item templates and to use their.


== Installation ==

1. Upload `animated_al_list` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create new project
4. Create a list with elements
5. Apply templates
6. Define items positions
7. Apply effect
8. Get codes from "Paste Code" tab and insert into page/post/template.

== Frequently Asked Questions ==

1.How can i modify template file?
Templates files you can find in "templates/front_templates" folder. It's consist of main and item. You can look at already exist files and create you own. Just don't forget to apply their in the list in "Apply Templates" tab.
